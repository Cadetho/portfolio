OHCC Circular Navigation Menu

Fall 2016

Created using D3.js this is a responsive and animated circular navigational menu for Open Health Care Clinic. It calculates label placement based upon label length and position on the circle.

Baton Rouge Provider Data Mapping

Jan 2017 - Present

This was a project in collaboration with the Together BR group to map different health statistics in Baton Rouge and surrounding areas. This data was used to define health districs and secure funding to offer medical services to areas that are underserved.
